## Usage Notes

These prompts are designed for Google Veo 3.1 and optimized for:
- **Resolution:** 4K (3840×2160) recommended
- **Duration:** 5-15 seconds per clip for b-roll, 15-30 seconds for hero sections
- **Style:** Cinematic, professional, futuristic with emerald/teal color palette
- **Mood:** Aspirational, innovative, trustworthy, efficient

For best results, append these modifiers to any prompt as needed:
- "seamless loop" for repeating backgrounds
- "slow motion" for dramatic effect
- "shallow depth of field" for focus emphasis
- "lens flare" for cinematic polish
- "4K, HDR" for maximum quality
